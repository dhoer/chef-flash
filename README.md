# Flash Cookbook

[![Cookbook Version](http://img.shields.io/cookbook/v/flash.svg?style=flat-square)][supermarket]
[![Build Status](http://img.shields.io/travis/dhoer/chef-flash.svg?style=flat-square)][travis]
